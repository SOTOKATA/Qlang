# Qlang

Qlang is a small programming language created as the final project of a student in a professional course. Language features:

- Working with the console
- Working with files
- Classes
- Functions
- Variables (arrays, numbers, strings)

[Guide book](https://sotokata.github.io/Qlang-guide-book/) to learn language

# Compile
1. Install newest package from this repo
2. Add path to ql to Enviroments PATH
3. To compile use follow command (in console): `ql [path]`
